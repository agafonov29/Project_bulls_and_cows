
Проект "Быки и Коровы"

Описание:
Это текстовая игра "Быки и Коровы", в которой игрок должен угадать скрытое число, состоящее из уникальных цифр. После каждого ввода система сообщает, сколько цифр игрок угадал правильно и в правильном месте (быки), а сколько цифр правильно угадано, но в неправильном месте (коровы).

Стек технологий:
В этом проекте использованы следующие технологии:
- Python 3.x

Установка:
Чтобы начать работу с этим проектом, выполните следующие шаги:

1. Клонируйте репозиторий:
   git clone https://github.com/agafonov29/Project_bulls_and_cows.git

2. Перейдите в каталог проекта:
   cd быки-и-коровы

3. Создайте и активируйте виртуальное окружение (рекомендуется для изоляции зависимостей):
   - Для Linux/macOS:
     python3 -m venv venv
     source venv/bin/activate
   - Для Windows:
     python -m venv venv
     venv\Scripts ctivate

4. Установите зависимости:
   Все необходимые библиотеки указаны в файле requirements.txt. Чтобы установить их, выполните команду:
   pip install -r requirements.txt

Запуск:
1. Запустите проект:
   После установки зависимостей, чтобы запустить игру, выполните:
   python gui.py

2. Следуйте инструкциям на экране, чтобы начать игру.

Структура проекта:
быки-и-коровы
    ├── __init__.py        # Инициализатор пакета
    ├── main.py            # Основной файл игры
    ├── gui.py             # Графический интерфейс
    ├── test_pytest.py     # Тестирование основновного кода
    ├── requirements.txt   # Файл зависимостей
    └── README.txt         # Этот файл

Как использовать:
- Запустите игру с помощью команды:
   python gui.py
- Игрок будет вводить число, и система будет выводить количество быков и коров для каждого ввода.
- Игра продолжается до тех пор, пока игрок не угадает правильное число.
